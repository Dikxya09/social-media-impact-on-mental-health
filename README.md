# Impact of Social Media on Student Mental Health

## Overview
This project analyzes the relationship between social media usage and students' mental health using data analysis and machine learning techniques. It explores how factors like screen time, sleep, and lifestyle habits influence stress, anxiety, and depression.

---

## Objectives
- Analyze the impact of social media usage on mental health  
- Identify key factors affecting stress, anxiety, and depression  
- Build a machine learning model to predict depression  

---

## Dataset Features
- Age, Gender  
- Daily Social Media Usage  
- Sleep Hours  
- Screen Time Before Sleep  
- Academic Performance  
- Physical Activity  
- Social Interaction Level  
- Stress, Anxiety, Addiction Levels  
- Depression Label  

---

## Data Analysis (EDA)
- Checked and handled missing values  
- Performed statistical analysis  
- Explored relationships between variables  
- Grouped data for deeper insights

## Machine Learning
- Models Used:
  - Random Forest  
  - Logistic Regression  

- Steps:
  - Data preprocessing & encoding  
  - Train-test split  
  - Model training  
  - Prediction and evaluation  

- Evaluation Metrics:
  - Accuracy  
  - Classification Report  

---

## Key Insights
- Higher social media usage is linked to increased stress and anxiety  
- Lower sleep duration increases the risk of depression  
- Balanced lifestyle habits improve mental well-being  

---

## Conclusion
The analysis shows that excessive social media usage negatively impacts students' mental health. Proper time management, reduced screen exposure, and healthy sleep patterns can help improve overall well-being.
